Focus is a simple macOS menu bar app that helps you stay consistent with your daily work. It is built with SwiftUI and runs as a lightweight Swift Package Manager project.

|**Features**|
- Lives in menu bar, one click away
- Appears on every window
- Add and check off daily tasks
- Completed tasks turn green and move to history
- Tracks daily streak
- Resets automatically each day
- Streak colors change at milestones (Orange, Blue at 30, Purple at 90, Red at 360)

|**Tools**|
- SwiftUI
- MenuBarExtra
- UserDefaults with JSON
- SF Symbols

Focus Tracker is a simple macOS menu bar app that helps you stay consistent with your daily work. It is built with SwiftUI and runs as a lightweight Swift Package Manager project.

What it does

It lives in your menu bar, always one click away.
You add tasks for the day and check them off as you finish them.
Completed tasks turn green and move to history.
Your daily streak shows how many days in a row you stayed productive.
Each day resets automatically, so you start fresh.

How to run it

Clone the repo.
Open Package.swift in Xcode.
Select the FocusApp target and press Cmd and R.

You will see the icon in your menu bar. Click it to manage your tasks.

Tech

SwiftUI, MenuBarExtra, UserDefaults with JSON, SF Symbols.
